# FortSmart Educacional - Landing Page

Bem-vindo ao repositório da página institucional do **FortSmart**, uma landing page moderna e educacional voltada para o contexto escolar técnico agrícola.

## 🚀 Tecnologias Integradas

- **HTML5**: Semântico e totalmente acessível, sem uso construtivo de frameworks pesados.
- **CSS3 Moderno**: Utilizando Flexbox, CSS Grid, variáveis nativas para Design System (cores, espaçamentos) e transições / animações elegantes com `cubic-bezier`. Parallax implementado para a Hero Section.
- **JavaScript Vanilla**: Scripts puros focados em performance, com o principal uso voltado ao `IntersectionObserver` que orquestra as animações ao rolar a página (Fade-in e Slide-up intercalados).

## 🛠 Como Publicar e Hospedar Gratuitamente (GitHub Pages)

A estrutura já está adequada para ser publicada em poucos minutos através do **GitHub Pages**.

### Passos de publicação:

1. **Criar Repositório:** Acesse sua conta e crie um novo repositório público no GitHub (ex: `fortsmart-institucional`).
2. **Enviar Arquivos:** Faça o push de todos os arquivos deste projeto (ou arraste e solte através da própria interface Web do GitHub).
3. **Ativar o GitHub Pages:**
   - Acesse a aba **Settings** do repositório.
   - Na barra lateral esquerda, clique em **Pages**.
   - Abaixo de "Build and deployment / Source", selecione a opção **Deploy from a branch**.
   - Em **Branch**, selecione `main` e deixe a pasta como `/(root)`.
   - Clique em **Save**.
4. **Finalização:** Em poucos minutos o link no topo contendo o endereço `seunome.github.io/nome-do-repositorio` estará disponível online!

## ✨ Destaques de Design e UX

- **Hero Parallax**: Experiência rica implementada nativamente mantendo uma velocidade responsiva alta na landing page.
- **Evolução da Timeline**: Construção vertical que se adapta perfeitamente em telas móveis e desktop, representando claramente a imersão em "Agricultura de Precisão".
- **Responsive-first**: A fonte redimensionável dinamicamente com clamps e blocos flexíveis para uma visualização excelente em SmartPhones iOS e Android.
